What is BuzzwordReader

Many buzzwords emerged in China every year on the Internet and buzzwords have become a part of daily communication among young people. BuzzwordReader is a Chrome Extension for searching, sharing, and defining Internet buzzwords. It aims to help users quickly understand popular buzzwords when browsing the Internet.

Check me on Github

How It Works

All interpretations of buzzwords come from users. First, similar to Wikipedia, users create an entry for each buzzword and then add an interpretation to it with a reliable source. Then, this entry would be able to found by other users.

But, how to ensure the quality and correctness of the interpretations of each entry? We introduced a rating mechanism. You can click the up arrow button if you think the interpretation is helpful. Otherwise, click the down arrow. Eventually, the interpretation with most agree appears on the top of an entry.

Search

You can simply highlight a word on any webpage and click the green button beside to search for related entries. A popup window with all related entries will show up.

Intepretation

In an entry, if you there is no interpretation yet, or existed interpretations are not correct, you can create one and add it into the entry.

Register

Before you can add an interpretation or vote for any existed interpretation, you need to create an account. In the setting page, you can create an account.
